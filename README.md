![Gameplay](assets/images/gameplay.gif)

Javascript implementation of retro game **Super Mario Bros**  

# Controls
Keys  | Action
------------- | -------------
Space, W  | Jump
ArrowLeft, A  | Move Left
ArrowRight, D | Move Right
M | Toggle Mute
\+ | Increase volume (10 level)
\- | Decrease volume (10 level)
